Aks-iom AE Plugins Installer

Автоматизированный скрипт с графическим интерфейсом для быстрой загрузки и установки популярных плагинов для Adobe After Effects. Программа избавляет от необходимости ручного поиска, скачивания и распаковывания архивов, полностью автоматизируя весь процесс.
✨ Особенности
- Популярные плагины: Поддержка автоматической установки ключевых пакетов (Sapphire, BCC, RSMB, Trapcode Suite, uwu2x и других).
- Полная автоматизация: Самостоятельное скачивание архивов (через gdown) и их распаковка в нужные директории After Effects.
- Система проверок: Финальная валидация наличия файлов после завершения процесса установки для каждого выбранного плагина.
- Двуязычность: Интерфейс и логирование поддерживают русский и английский языки.
